# Color Palette

## Philosophy
**Color is used for meaning, not decoration.**

Muted, sophisticated, editorial. No loud gradients. No glossy effects.  
Authority through restraint.

---

## Primary Brand Colors

### 1. Authority Green (Primary)
```
Deep Forest Green / Dark Teal
HEX: #1A3A32
RGB: 26, 58, 50
HSL: 162, 38%, 16%

Use: Headlines, primary CTAs, trust elements, navigation active states
```

```
Muted Lime / Soft Neon Green
HEX: #C5F467
RGB: 197, 244, 103
HSL: 80, 86%, 68%

Use: Highlights, success states, AI features, "thinking" indicators
```

## Usage Rules

### ✓ DO
- [ ] Use Authority Green for primary actions
- [ ] Use Innovation Lime sparingly for impact
- [ ] Maintain 4.5:1 contrast ratio minimum
- [ ] Use semantic colors consistently
- [ ] Test colors in glassmorphism contexts

### ✗ DON'T
- [ ] Mix multiple accent colors in one view
- [ ] Use Innovation Lime for text
- [ ] Create custom gradients
- [ ] Use pure black (#000000)
- [ ] Add glow effects or neon treatments
