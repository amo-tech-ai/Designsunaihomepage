## Quick Reference

### Primary Colors
- **Authority Green:** `#1A3A32` (Deep forest green)
- **Innovation Lime:** `#C5F467` (Muted lime/soft neon)
- **Background Ivory:** `#FAF8F5` (Warm off-white)

### Typography
- **Headlines:** Editorial serif (Georgia, Playfair Display)
- **Body:** Humanist sans (Inter, SF Pro)
- **UI Text:** Neutral sans (Inter)

### Spacing Scale
`4px · 8px · 12px · 16px · 24px · 32px · 48px · 64px · 96px · 128px`

## Design Principles

### ✓ DO
- [ ] Use large whitespace
- [ ] Maintain editorial pacing
- [ ] Apply color for meaning, not decoration
- [ ] Design for decision-makers
- [ ] Create breathing room in layouts

### ✗ DON'T
- [ ] Use loud gradients
- [ ] Add glossy effects
- [ ] Include playful illustrations
- [ ] Overcrowd layouts
- [ ] Use harsh dividers
