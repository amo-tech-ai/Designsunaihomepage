## Design System

### Color Palette
- **Primary Green:** `#1A3A32` (Authority Green)
- **Text Dark:** `#1F2421`
- **Text Medium:** `#6B7370`
- **Border:** `#8B9D8A`
- **Background:** `#FAF8F5` (Warm White)
- **Accent Gradients:**
  - Emerald: `from-emerald-600 to-teal-500`
  - Rose: `from-rose-600 to-pink-500`
  - Purple: `from-purple-600 to-violet-500`
  - Orange: `from-orange-600 to-amber-500`
  - Blue: `from-blue-600 to-indigo-500`

### Typography
- **Headlines:** Font-light, large scale (text-4xl to text-6xl)
- **Body:** Font-light, leading-relaxed
- **Labels:** Uppercase, tracking-[0.2em], font-bold
- **Exhibits:** "EXHIBIT 1", "EXHIBIT 2", etc.
- **Section Numbers:** "01", "02", "03" prefixes

### Spacing
- **Section Padding:** `py-32` (128px vertical)
- **Container Max:** `max-w-[1400px]` or `max-w-[900px]` (article)
- **Grid Gaps:** `gap-6` to `gap-16`

### Animations
- **Scroll Progress:** Fixed top bar with scaleX transform
- **Parallax:** Y-axis transforms on scroll
- **Fade In:** opacity 0 → 1 with whileInView
- **Slide In:** Y/X offset with whileInView
- **Hover States:** Scale, translate, color changes
- **Delays:** Staggered (0.1s increments)

## Technical Implementation

### Dependencies
- `motion/react` - Animations and scroll effects
- `lucide-react` - Icon library
- Tailwind CSS v4 - Styling

### Performance Optimizations
- `viewport={{ once: true }}` - Animations trigger once
- `prefersReducedMotion` check - Respects user preferences
- Lazy-loaded images
- useMemo for complex calculations (dot matrix)

### Accessibility
- Semantic HTML structure
- ARIA labels where appropriate
- Keyboard navigation support
- Color contrast ratios (WCAG AA)
