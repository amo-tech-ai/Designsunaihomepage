## Animation Specs

### Timeline Reveal
```javascript
phases: 7 nodes on horizontal line
revealSequence: {
  stagger: 0.3s,
  each: {
    node: { scale: [0,1.2,1], opacity: [0,1] },
    label: { y: [10,0], opacity: [0,1], delay: 0.2s },
    connector: { scaleX: [0,1], duration: 0.4s }
  }
}
currentPhase: { 
  scale: [1,1.15,1], 
  boxShadow: pulse,
  loop: true,
  duration: 2s
}
```

### Phase Card Expansion
```javascript
card.onClick: {
  collapsed: { height: '80px' },
  expanded: { height: 'auto', duration: 400ms },
  content: {
    opacity: [0,1],
    y: [-10,0],
    stagger: 0.1s (activities, deliverables, criteria)
  },
  otherCards: { opacity: 0.5 }
}
```

### Deliverables Checklist

checkbox.onClick: {
