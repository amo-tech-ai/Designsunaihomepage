## FAQ Section

**Q1: What's the typical timeline?**
A: 8-12 weeks depending on complexity. Starter projects: 8 weeks. Professional: 10-12 weeks. Enterprise: 12-16 weeks.

**Q2: Do you integrate with existing systems?**
A: Yes. We integrate with Stripe, Salesforce, HubSpot, Slack, and custom APIs.

**Q3: What AI models do you use?**
A: OpenAI GPT-4 for text, Whisper for audio, and custom fine-tuned models for specialized tasks.

**Q4: Can you handle user authentication?**
A: Yes. Supabase provides email/password, magic links, OAuth (Google, GitHub), and SSO.

**Q5: Is the code white-label?**
A: Enterprise tier includes full white-label rights. Professional tier includes code ownership.

**Q6: What's included in support?**
A: Bug fixes, minor updates, and technical assistance. 30 days (Starter), 90 days (Pro), 24/7 (Enterprise).

**Q7: Do you offer ongoing maintenance?**
A: Yes. $2,500/month for updates, monitoring, and feature additions.

**Q8: Can I scale later?**
A: Absolutely. Built on Supabase/Vercel for automatic scaling. No infrastructure changes needed.
